## js-link-finder / LinkFinder

The endpoint-intelligence workflow is conceptually inspired by the passive
quoted-link discovery approach used by
[PortSwigger/js-link-finder](https://github.com/portswigger/js-link-finder),
which credits the original
[LinkFinder](https://github.com/GerbenJavado/LinkFinder) project. The
PortSwigger repository is distributed under the MIT License; its LICENSE file
carries copyright © 2019 InitRoot, while its README identifies Frans Hendrik
Botes and credits LinkFinder for the idea and regular expression.

JS Secret Hunter does not bundle or execute the Jython extension. Its endpoint
extractor is an independent TypeScript implementation designed for Caido with
bounded matching, call-site and method inference, template-route normalization,
static-asset and module exclusions, canonical route keys, deduplication,
source-line mapping, and sensitive-query redaction.
